# Audio Assets para Victoria en Cristo

Esta carpeta debe contener los siguientes archivos de audio:

## Música de Fondo (BGM)

Las pistas contextuales están en `assets/sounds/bgm/` y fueron seleccionadas
para que cada sección tenga una intención musical distinta:

- `bgm_home_peaceful_days.mp3` - Home / navegación de Escuela del Reino (calma luminosa)
- `bgm_reflection_place_home.mp3` - Biblia, oración, diario y lectura devocional (contemplativa)
- `bgm_focus_innocence.mp3` - Quizzes, Maná, orden bíblico y memorización (foco sin distracción)
- `bgm_story_summer_memories.mp3` - Recorridos, timeline, mapas y narrativa bíblica (aventura suave)
- `bgm_mystery_suspense.mp3` - Profecías y cumplimiento mesiánico (misterio breve)
- `bgm_battle_determined_pursuit.mp3` - Duelo Relámpago (tensión rápida / contrarreloj)
- `bgm_game_let_the_games_begin.mp3` - Juegos sociales como Headbanz y Carrera (energía ligera)

## Efectos de Sonido (SFX)
- `click.mp3` - Sonido corto para clicks de botón (~0.1s)
- `success.mp3` - Sonido de logro/éxito (~0.5s)
- `select.mp3` - Sonido sutil para selecciones (~0.1s)

## Recomendaciones
- Usar formato MP3 para compatibilidad
- Mantener archivos pequeños (<500KB para SFX, idealmente <3MB para BGM)
- BGM debe ser un loop seamless (sin cortes abruptos)
