Para reemplazar los placeholders con ilustraciones reales:

1. Generar imágenes con DALL-E 3 usando los prompts en `PLAN_COVERS_SPECS.md`
2. Descargar como JPG/PNG de 1024x1536px mínimo
3. Nombrar con el formato: `<plan-id>.png`
4. Colocar en `assets/images/plan_covers/`
5. Hot reload
